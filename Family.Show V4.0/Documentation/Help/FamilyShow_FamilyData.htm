Family Data

This screen presents a summary of the all family information in an editable list view style.

Data panels

In the right hand side of the view, several statistical panels are shown including Surnames, Age Distributions, Living Status, Gender and Birthdays. Select an item in a panel to filter the list view.

List View

The main section of the screen contains a list view of the family tree data. The displayed data can be filtered.

Supported Filters Usage
[!][filter] [filter] = photos, attachment, note, citations, restriction e.g. photos returns all people with photos, !note returns all people with no notes.
[living] or [deceased] e.g. living returns all living people.
[male] or [female] e.g. female returns all living people.
[age]-[age] For age ranges e.g. 0-16 returns people ages 0 to 16 years.
[age]+ For age ranges e.g. 64+ returns people over 64.
[dd]/[mm] or [mm]/[yyyy] For dates of birth e.g. 01/10 returns people born on 1st October or 10/1999 returns people born in October 1999.


To view additional columns in the list view, right click over the list view area and select additional columns from the menu list. Columns can be dragged to reorder and clicked to sort.
