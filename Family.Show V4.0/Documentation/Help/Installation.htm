Installation

System Requirements

Family.Show can be installed on the following operating systems:

  • Windows XP*
  • Windows Vista*
  • Windows 7**

    * .Net 3.5 SP1 or higher is needed to run Family.Show in Windows XP and Windows Vista. In Windows 7, the required .Net components are installed with the operating system.
    **Features such as Jump Lists and Touch are only available in Windows 7.

    To install Family.Show

    To install Family.Show you can use the ClickOnce deployment from the CodePlex website.

    To remove Family.Show

    Family.Show can be removed using the Add/Remove Programs options in the Control Panel.
