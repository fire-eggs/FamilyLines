Merge

This option can be used merge people from another tree file into the current tree file. After importing new people, use the Create Relationship option to connect the imported people with people in the original tree. To help, you are provided with the option to append * to the names of imported people so that they can be easily filtered in any list. Sources and repositories are also checked for duplication. New sources and repositories are imported and duplicate sources and repositories are merged. After the merge, a comparison of duplicate people is displayed to allow update of duplicated people.
