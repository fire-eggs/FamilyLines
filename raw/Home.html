Project Description

Family Lines is a free, open-source professional level genealogy application that allows you to easily track, organize, and share your family history.

This project is a re-launch of the "Family.Show" project.

Our goals:

  • to provide support to the "Family.Show" user community
  • to extend current features in "Family.Show"
  • to add additional features requested by the "Family.Show" user community
  • to add new features: complete GEDCOM support; charts; reports

SVN URL:

How to access a project's SVN repository is one of the more poorly documented aspects of Codeplex. The Family Lines SVN repository URL is:

https://familylines.svn.codeplex.com/svn

You don't need to be a project member to get the code. Credentials should not be required for read access. If you have problems, contact us or CodePlex support!
